## About

Template Screen is a showcase of professionally designed landing page templates built with Astro 5. Each template demonstrates a unique design aesthetic and can be customized to meet specific client needs. The project features a modern catalog interface with real-time search and tag-based filtering capabilities. Perfect for web designers, developers, and agencies looking to showcase various design approaches to their customers.

## Development Guidelines

- Use Astro for building components and pages
- Follow the existing folder structure for consistency
- Keep components modular and reusable
- Use smooth, professional animations
- Ensure responsive design across all templates
- Use Bun as the package manager (not npm)
